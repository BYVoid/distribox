void Distribox.FileSystem.FileWatcher.Dispose ( )
inline

Releases all resource used by the Distribox.FileSystem.FileWatcher object.

Call Dispose when you are finished using the Distribox.FileSystem.FileWatcher. The Dispose method leaves the Distribox.FileSystem.FileWatcher in an unusable state. After calling Dispose, you must release all references to the Distribox.FileSystem.FileWatcher so the garbage collector can reclaim the memory that the Distribox.FileSystem.FileWatcher was occupying.
