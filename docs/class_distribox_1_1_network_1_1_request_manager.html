Detailed Description

The manager of sending patches RequestManager will

  • Decide which Patches to request right now
  • Manage request to ensure no request will be sent twice
  • Resend request if request isn't finished It maintains two queues in it: TODO queue and Doing queue, the lifecycle of a request is getRequests FinishRequests TODO ----------—> Doing -------------—> Done -------------—> Fail / Expire
